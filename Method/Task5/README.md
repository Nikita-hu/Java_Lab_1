# Равенство

## Описание
Дана сигнатура метода: 
```java
public bool isEqual(int a, int b, int c)
```
Необходимо реализовать метод таким образом, чтобы он возвращал true, если
все три полученных методом числа равны

## Пример 1:
a=3 b=3 с=3
**результат:** true

## Пример 2:
a=2 b=15 с=2
**результат:** false
