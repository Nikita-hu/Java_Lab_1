# Все вхождения

## Описание
Дана сигнатура метода: 
```java
public int[] findAll (int[] arr, int x)
```
Необходимо реализовать метод таким образом, чтобы он возвращал новый
массив, в котором записаны индексы всех вхождений числа x в массив arr.

## Пример:
arr=[1,2,3,8,2,2,9]
x=2
**результат:** [1,4,5]
